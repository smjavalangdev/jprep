#### Java 8 Features


- Default methods for interfaces
- Lambda Expressions
- Functional interfaces
- Method and constructor references
- Lambda Scope
- Stream processing
- Parallel Streams
- New Date API
- Annotations
- Nashorn ( Embedded javascript engine ) - allows you to evaluate Javascript on the JVM dynamically
